Azure Media Cognitive Demo: Video OCR and Search

Video Track




Key technologies and features:

  • Video OCR: Text is displayed on the video closed caption above as the same visual text acutally appear on the video (see note below). This is achieved by Azure Media Services OCR Media Processor(MP) which can converts visual text from video into digital text. [note] It's configured in this demo that 1/4 region from top is exluded from converting visual text target region on the video.
  • Text Search: You can search for words or phrases on visual text appears on the video by typing keywords on the search form above. This is achieved by Azure search which indexes digital text generated from Video OCR and makes them searchable
